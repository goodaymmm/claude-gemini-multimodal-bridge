MCP Router for Claude Code ↔ Gemini CLI ↔ Google AI Studio

CGMB

Claude-Gemini Multimodal Bridge — search, reason, and generate (image/audio/PDF) in one workflow.

GitHub

Install

npm install -g claude-gemini-multimodal-bridge

What it does

  • Gemini CLI → web search & quick retrieval
  • Google AI Studio → multimodal generation (image/audio) + document handling
  • Claude Code → deep reasoning and coding workflows
